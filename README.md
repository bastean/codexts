## Run

Install required

- [Node.js/npm](https://nodejs.org/en/download)

### Locally

- HTTPS

  ```bash
  git clone https://github.com/bastean/codexts.git && cd codexts/ && npm run init && npm start
  ```

- SSH

  ```bash
  git clone git@github.com:bastean/codexts.git && cd codexts/ && npm run init && npm start
  ```
